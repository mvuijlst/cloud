# Pipe 2

An endless self-playing isometric pipe runner built from the original pixel-art sprites in [../pipe/images](../pipe/images).

## Controls

- `A` pause or resume autoplay
- `R` restart the run

## Notes

- There is no end tile; the board scrolls with the live flow.
- Future tiles are planned ahead automatically.
- Liquid is rendered as one continuous path so it stays smooth across tile boundaries.
